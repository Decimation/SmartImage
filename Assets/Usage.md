# Command-Line

| Parameter | Definition                                            | Type       |
| --------- | ----------------------------------------------------- | ---------- |
| `-i`      | Input                                                 | _`value`_  |
| `-auto`   | Start search immediately                              | _`switch`_ |
| `-h`      | Hide GUI                                              | _`switch`_ |
| `-s`      | Switch current queue input to value specified by `-i` | _`switch`_ |
